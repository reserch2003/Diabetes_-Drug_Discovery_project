# Diabetes_-Drug_Discovery_project
#In-Silico Drug Discovery Against Diabetes Target Proteins

This project focuses on molecular docking and bioinformatics-based screening of anti-diabetes- related proteins using PyRx and Discovery Studio.

Tools used-
PyRx
BIOVIA Discovery Studio 
AutoDock Vina
